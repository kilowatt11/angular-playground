Bindings

So we have learned that using the double curlies {{ }} allows us to run javascript code directly from our html. We also know that we have access to our controller object and can get its properties to render on the page as well.

Another method of bring in properties from our controller is to use another ng- directive called ng-model

For example

  • Binding our Componentr and View

    
<input ng-model="component.name" />
<div>
	<p>
		Hello {{component.name}},
	</p>
</div>

  • Go ahead and edit the input field and watch what happens ...

    Hello {{component.name}},

    Isn't Angular impressive the way it can bind your view and your javascript with such little code? No need to call update() anywhere because the view is already bound to the property of the object. When the JavaScript changes the view changes.
